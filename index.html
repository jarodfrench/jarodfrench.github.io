Introduction:¶

Lionel Messi is one of the best soccer players on the planet. There has been a long debate between soccer fans whether the greatest player of all time is Messi, or if it is Cristiano Ronaldo, Pelé, the late Diego Maradona, or others. In this analysis, I will be looking at all of the goals Messi has scored in his club soccer career.

I will be analyzing all of Messi's goals by following the Data Science Pipeline:

  1. Data collection/curation + parsing (if necessary)
  2. Data management/representation
  3. Exploratory data analysis
  4. Hypothesis testing
  5. Communication of insights attained

I will be using this dataset of all of Messi's club soccer goals, found on Kaggle. The dataset includes data such as the season, competition, matchday, venue, date, Messi's team, the opponent, the result, Messi's position, the minute the goal was scored, the score when Messi scored the goal, the type of goal, and who assisted the goal. The dataset was last updated April 23, 2022 so there may be a few missing goals that Messi has scored between then and now. So with that, consider the data collected.

Lets calculate the actual average goals Messi scored over his whole career and over the 08/09 to 20/21 seasons.

# obtain a list of each season Messi has played
seasons = messi['Season'].unique()

# make a list of DataFrames, grouped by season
seasons_list = messi.groupby('Season')

# The length of each DataFrame in seasons_list is the number of Goals Messi scored that season
lengths = []
for season in seasons:
    lengths.append(len(seasons_list.get_group(season).index))

# sum the total amount of goals Messi has scored and divide it by the number of total seasons
# number of total seasons is the length of the list of unique seasons in the seasons variable 
sum = 0
for season in lengths:
    sum += season
print("Messi scored approximately " + str(sum / len(lengths)) + " goals per season from 04/05 to 21/22.")

# sum the seasons from 04/05 to 20/21 in a similar way
sub_sum = 0
# 4 and 16 are the indices of lengths here that correspond to 04/05 and 20/21 respectively
for i in range(0,len(lengths)):
    if i >= 4 and i <= 16:
        sub_sum += lengths[i]
# divide by 13 because 04/05 to 20/21 is 13 seasons (inclusive)
print("Messi scored approximately " + str(sub_sum / 13) + " goals per season from 08/09 to 20/21.")

It is pretty obvious how many goals Messi has scored for Barcelona since La Liga, Copa del Rey and most of Messi's UEFA Champions League goals were scored while he was at Barcelona. That makes this a very biased plot, so to offset the amount of time Messi was playing at Barcelona, we are going to standardize the goals in each competition to the number of goals per game in each competition. # dataset = data to use, group_names are the names of the chunks,
# column_name is the name of the column to sort by to get the chunks
def cut_dataframe_into_chunks(dataset, group_names, column_name):
    # with the data cut into the number of bins, append each bin into a list of datasets based on the groups above
    tables = []
    for curr in group_names:
        tables.append(dataset.groupby(column_name).get_group(curr))
    return tables

group_names = messi['Competition'].unique()
grouped_messi = cut_dataframe_into_chunks(dataset=messi, group_names=group_names, column_name='Competition')

Now we have all of the individual tables, one for each competition. We are in need of the total amount of games Messi has played in each competition in order to do this. Since our dataset only has the games in which Messi scores, it does not include games in which he does not score, so we will be relying on another dataset. This one is from FBref, which allows people to use the data for free as long as we mention that we got the data from them, and only if we are not creating something to rival FBref. Thank you FBref! (data found at this link: https://fbref.com/en/players/d70ce98e/all_comps/Lionel-Messi-Stats---All-Competitions) However the FBref data does not include all the years of the Supercopa de Espana, the FIFA Club World Cup or the Copa del Rey, so they will be ommitted from the goals per game and minutes per goal analysis.

Now that we have another dataset, we need to clean this one up a little bit. We don't need most of the columns, so we are just going to keep the Season, Squad, Comp, MP, Min, and 90s column, so we can drop the rest of them using Pandas drop function. Another thing we need to clean is that in the Comp column, competitions include "1." in front of them, so to make it easier to read, we are going to replace the "1." with an empty string, effectively getting rid of it.
